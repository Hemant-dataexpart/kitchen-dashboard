## 🚀 Getting Started With kitchen-dashboard

After installation, you can start using kitchen-dashboard right away. This section explains how to connect it to your Home Assistant system and customize it.

### Connect to Home Assistant

1. Launch the kitchen-dashboard app.

2. The first screen will ask for your Home Assistant URL. This is usually something like `http://homeassistant.local:8123` or your device’s IP address followed by port 8123.

3. Enter your Home Assistant URL and press "Connect."

4. The app will request your login credentials to access Home Assistant. Enter your username and password.

5. If connection works, kitchen-dashboard will load your Home Assistant settings and devices.

### Customize Your Dashboard

- Use the settings menu inside the app to select which sensors, switches, and controls appear on your wall tablet.

- Adjust the layout to fit your vertical screen for easy viewing.

- Add your favorite cards like weather, calendar, media controls, or lights.

- Save your configuration to keep it ready for daily use.

### Using the Dashboard

- The dashboard shows live updates from your Home Assistant devices.

- Touch controls respond quickly to turn on lights, check sensors, or play media.

- Swipe through pages or sections as organized.

- The screen stays awake during use to avoid interruptions.

## 🛠 Troubleshooting Tips

If you run into issues, try the following:

- Make sure your computer and the tablet are on the same Wi-Fi network as your Home Assistant server.

- Verify the Home Assistant URL is correct and accessible from your device.

- Restart the kitchen-dashboard app to refresh the connection.

- Check for updates on the release page and download the latest version if needed.

- If the app does not start, try reinstalling it.

- Confirm your Windows firewall or security software allows kitchen-dashboard to access the network.
